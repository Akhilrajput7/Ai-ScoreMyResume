# 🚀 AI Score My Resume

> An AI-powered Resume ATS Analyzer that evaluates resumes, provides ATS scores, identifies skill gaps, generates personalized learning roadmaps, and creates interview preparation plans using Generative AI.
> 
---

## 📌 Overview

AI Score My Resume is an intelligent career assistant designed to help students, freshers, and professionals optimize their resumes for Applicant Tracking Systems (ATS).

Unlike traditional ATS checkers, this platform leverages Generative AI to analyze resumes in-depth, identify missing technical skills, compare resumes against job descriptions, and generate personalized learning roadmaps and interview preparation plans.

The goal is to bridge the gap between a candidate's current skills and the skills required for their dream job.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Akhilrajput7/Ai-ScoreMyResume.git
```

```
cd Ai-ScoreMyResume
```

## Install Frontend

```bash
cd Frontend
npm install
```

## Install Backend

```bash
cd Backend
npm install
```

---

# Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000

MONGO_URI=your_mongodb_url

JWT_SECRET=your_secret_key

GEMINI_API_KEY=your_api_key
```

---

# Run Project

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```
